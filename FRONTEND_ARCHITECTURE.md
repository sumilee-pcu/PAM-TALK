# PAM-TALK 프론트엔드 아키텍처 계층도

## 🏗️ 시스템 개요

```
PAM-TALK Platform
├── 👔 ESG Committee Portal (위원회 포털)
│   └── 쿠폰 발행 및 MRV 검증
│
├── 🛠️ Admin Dashboard (운영자 대시보드)
│   └── 시스템 관리 및 모니터링
│
└── 👥 PAM-TALK User Portal (사용자 포털)
    └── 일반 사용자용 서비스
```
