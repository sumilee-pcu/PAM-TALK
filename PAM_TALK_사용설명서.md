# PAM-TALK 플랫폼 사용 설명서

## 📱 플랫폼 소개

PAM-TALK는 **Platform for Active Meta**의 줄임말로, 지역 농수축산물 유통 혁신을 위한 통합 소셜 플랫폼입니다. 농촌과 도시를 연결하여 탄소발자국 감소와 지역 경제 활성화를 동시에 달성하는 혁신적인 생태계를 제공합니다.

## 🌐 접속 방법

**메인 웹사이트**: [https://pam-talk.vercel.app](https://pam-talk.vercel.app)

- 데스크톱, 태블릿, 모바일 모든 기기에서 접속 가능
- 별도 앱 설치 없이 웹브라우저로 바로 이용

## 👤 회원가입 및 로그인

### 회원 유형 선택
1. **농부 (Farmer)**: 농산물 생산자
2. **소비자 (Consumer)**: 일반 구매자
3. **유통업체 (Distributor)**: 중간 유통업체
4. **식당 (Restaurant)**: 음식점 사업자

### 가입 절차
1. 메인페이지에서 "회원가입" 클릭
2. 회원 유형 선택
3. 기본 정보 입력
   - 사용자명
   - 지역 정보 (지역, 도시)
   - 프로필 정보
4. 계정 생성 완료

## 🎯 핵심 기능 사용법

### 1. 🌍 탄소발자국 추적 시스템

#### 개인 대시보드
- **위치**: 메인 화면 상단 "My Carbon" 섹션
- **확인 가능 정보**:
  - 개인별 누적 탄소 절약량
  - 에코 포인트 잔액
  - 이번 달 환경 기여도

#### 탄소 절약 활동 기록
1. **로컬 푸드 구매**: 지역 농산물 구매 시 자동 기록
2. **친환경 배송**: 근거리 배송 선택 시 보너스 포인트
3. **재사용 포장**: 친환경 포장재 사용 시 추가 포인트

### 2. 📱 소셜 네트워킹 기능

#### 피드 이용법
- **홈 피드**: 팔로우한 사용자들의 최신 활동
- **지역 피드**: 같은 지역 사용자들의 게시물
- **트렌딩**: 인기 있는 농산물 및 토픽

#### 게시물 작성
1. "새 게시물" 버튼 클릭
2. 농산물 사진 업로드
3. 설명 및 해시태그 추가
4. 가격 및 판매 정보 입력 (농부의 경우)
5. 게시물 발행

#### 상호작용
- **좋아요**: 하트 아이콘 클릭
- **댓글**: 게시물 하단에 의견 작성
- **공유**: 내 네트워크에 게시물 공유

### 3. 🛒 직거래 마켓플레이스

#### 상품 검색
- **지역별 검색**: 거주 지역 기준 농산물 찾기
- **카테고리 필터**: 과일, 채소, 곡물 등 분류별 검색
- **가격대 필터**: 예산에 맞는 상품 검색

#### 주문 과정
1. 원하는 농산물 선택
2. 수량 및 배송 옵션 선택
3. 농부와 직접 채팅으로 상담
4. 주문 확정 및 결제
5. 배송 추적

#### 농부 판매 가이드
1. **상품 등록**:
   - 상품명, 설명, 가격 입력
   - 고품질 사진 업로드
   - 재배 방법, 수확 시기 명시
2. **주문 관리**:
   - 실시간 주문 알림 확인
   - 고객과의 소통
   - 배송 일정 조율

### 4. 🏆 리워드 시스템

#### 에코 포인트 적립 방법
- **로컬 푸드 구매**: 구매 금액의 5% 포인트 적립
- **친환경 활동**: 탄소 절약 활동 시 보너스 포인트
- **커뮤니티 참여**: 리뷰 작성, 추천 등으로 포인트 획득
- **지속적인 참여**: 연속 이용 시 누적 보너스

#### PAM 토큰 시스템
- **토큰 획득**: 에코 포인트와 1:1 교환
- **토큰 사용**:
  - 농산물 할인 구매
  - 프리미엄 기능 이용
  - 지역 이벤트 참여권 구매

## 🌱 탄소 챌린지 참여 방법

### 월간 챌린지
1. **로컬 푸드 챌린지**: 한 달간 50% 이상 지역 농산물 구매
2. **제로 웨이스트 챌린지**: 일회용품 사용 최소화
3. **친환경 배송 챌린지**: 친환경 배송 옵션만 선택

### 챌린지 참여 절차
1. 메인 화면 "챌린지" 메뉴 클릭
2. 참여하고 싶은 챌린지 선택
3. "참여하기" 버튼 클릭
4. 일일 활동 기록
5. 목표 달성 시 보상 획득

## 🔗 블록체인 기능

### Algorand 연동 혜택
- **투명한 거래**: 모든 거래 내역이 블록체인에 기록
- **신뢰성 보장**: 변조 불가능한 거래 증명
- **빠른 처리**: 4.5초 이내 거래 완료
- **저렴한 수수료**: 거래당 약 $0.0003

### ESG 토큰 활용
- **환경 기여 증명**: 탄소 절약 활동의 블록체인 인증
- **토큰 거래**: P2P 토큰 교환 및 거래
- **투명한 이력**: 모든 환경 기여 활동 추적 가능

## 📍 지역별 커뮤니티

### 지역 그룹 가입
1. 프로필에서 거주 지역 설정
2. 해당 지역 커뮤니티 자동 가입
3. 지역별 농산물 정보 및 이벤트 알림 받기

### 지역 특화 기능
- **계절별 농산물 캘린더**: 지역 특산품 수확 시기 안내
- **농가 직접 방문**: 농장 체험 및 직접 구매 일정
- **지역 축제 연동**: 농업 관련 지역 행사 정보

## ⚙️ 계정 설정

### 프로필 관리
- **프로필 사진**: 본인 또는 농장 사진 업로드
- **소개글**: 농업 철학 또는 구매 성향 작성
- **관심 카테고리**: 선호하는 농산물 종류 설정

### 알림 설정
- **새 게시물 알림**: 팔로우 농부의 새 상품 알림
- **가격 변동 알림**: 관심 상품 가격 변동 시 알림
- **챌린지 알림**: 참여 중인 챌린지 진행 상황 알림

### 개인정보 보호
- **위치 정보**: 정확한 주소 대신 지역 단위로만 공개
- **연락처 보호**: 플랫폼 내 메시징 시스템 이용
- **거래 정보**: 개인별 거래 내역 암호화 저장

## 🆘 고객 지원

### 자주 묻는 질문 (FAQ)
**Q: 에코 포인트를 현금으로 환전할 수 있나요?**
A: 에코 포인트는 플랫폼 내에서만 사용 가능하며, PAM 토큰으로 교환하여 농산물 구매에 활용할 수 있습니다.

**Q: 농산물 품질에 문제가 있을 때는 어떻게 하나요?**
A: 상품 수령 후 3일 이내에 사진과 함께 문제를 신고하면, 농부와의 중재를 통해 해결책을 제시합니다.

**Q: 농부 인증은 어떻게 이루어지나요?**
A: 농업 관련 서류(농업인증서, 사업자등록증 등) 제출을 통해 인증하며, 블록체인을 통해 신원이 검증됩니다.

### 고객센터 연락처
- **이메일**: support@pamtalk.co.kr
- **플랫폼 내 1:1 문의**: 설정 → 고객센터 → 문의하기
- **응답 시간**: 평일 09:00-18:00 (24시간 이내 답변)

## 🚀 시작하기 팁

### 농부를 위한 팁
1. **고품질 사진 업로드**: 자연광에서 촬영한 신선한 농산물 사진
2. **상세한 설명 작성**: 재배 방법, 농약 사용 여부, 보관법 등 상세 기재
3. **정기적인 소통**: 고객과의 적극적인 소통으로 신뢰 관계 구축
4. **계절별 상품 기획**: 시즌에 맞는 농산물 패키지 제안

### 소비자를 위한 팁
1. **지역 농부 팔로우**: 거주 지역 인근 농부들을 팔로우하여 신선한 정보 획득
2. **리뷰 활발히 작성**: 구매 후 리뷰 작성으로 포인트 적립 및 농부 지원
3. **계획적인 구매**: 계절별 농산물 캘린더 참고하여 제철 농산물 구매
4. **커뮤니티 참여**: 농업 관련 토론 및 정보 공유로 더 나은 선택
